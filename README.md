# KI-Biomedizinische Signalanalyse

Dieses Projekt befindet sich im Aufbau.

Ziel ist die Entwicklung einer datengetriebenen Analysepipeline für physiologische Signaldaten. Der Fokus liegt auf Signalverarbeitung, Datenvisualisierung, Machine Learning und statistischer Analyse mit Python und R.

## Projektziele

- Verarbeitung physiologischer Signaldaten
- Visualisierung medizinischer Zeitreihen
- Signalfilterung und Feature-Extraktion
- Anwendung einfacher Machine-Learning-Methoden
- Ergänzende statistische Analyse mit R

## Technologien

- Python
- Pandas, NumPy, SciPy
- Matplotlib, Plotly
- scikit-learn
- R
- Git und GitHub

## Status

Aktuell in Entwicklung.
